General Questions

What is LearnDocs Hub?

LearnDocs Hub is a free collection of 100+ online tools for working with PDFs, images, text, audio, video, and business documents. It also includes a free AI Chat assistant powered by Groq. All tools run entirely in your browser - your files never leave your device.

Is it really free? Are there any hidden charges?

Yes, completely free! All 100+ tools are free to use with:

  • No hidden charges or subscription fees
  • No watermarks on your files
  • No usage limits
  • No account creation required
Do I need to install any software?

No installation required! All tools work directly in your web browser. Just open the website and start using any tool immediately. Works on Chrome, Firefox, Safari, Edge, and other modern browsers.

Can I use LearnDocs Hub on mobile?

Yes! The website is fully mobile-responsive and works on smartphones and tablets. Many tools also support camera input on mobile devices for scanning documents or capturing images.

Privacy & Security

Are my files uploaded to a server?

No! All file processing happens locally in your browser using JavaScript. Your files are never uploaded to any server. This makes our tools one of the most private options available online. Your data stays on your device.

Is my data safe?

Absolutely. Since files are processed locally in your browser, there's no risk of your documents being stored or accessed by anyone else. We cannot see or access your files at all.

Do you track user activity?

We use basic analytics to understand which tools are popular, but this is anonymized. We don't track personal information, and no file data is ever sent to our servers.

Yes, the AI Chat is completely free with no limits. You can have unlimited conversations with the AI assistant at no cost.

What can I ask the AI assistant?

You can ask the AI to help with:

  • Writing emails, essays, and documents
  • Programming and coding help

Tools & Usage

What types of tools are available?

We offer 100+ tools across categories:

  • PDF Tools - merge, split, compress, convert, edit, annotate
  • Image Tools - resize, crop, convert, compress, watermark
  • Text Tools - QR generator, password generator, formatters
  • Audio Tools - convert, trim, merge, extract audio
  • Video Tools - compress, convert, trim, create GIFs
  • Business Tools - invoices, resumes, signatures
  • Calculators - age, BMI, unit conversion
  • Social Media Tools - thumbnails, post creators
What file formats are supported?

Different tools support different formats:

  • PDF tools: PDF, DOCX, XLSX, PPTX, and images
  • Image tools: JPG, PNG, WebP, GIF, BMP, SVG
  • Audio tools: MP3, WAV, OGG, AAC, FLAC
  • Video tools: MP4, MOV, AVI, WebM, and more
Is there a file size limit?

Most tools work with files up to 50MB. Since files are processed in your browser, larger files may take longer to process depending on your device's capabilities.

What happens if a tool doesn't work?

Since all processing happens in your browser, some tools may have limitations based on browser support. For best results, use a modern browser like Chrome, Firefox, or Edge. If a tool isn't working, try:

  • Refreshing the page
  • Clearing your browser cache
  • Trying a different browser
  • Checking if JavaScript is enabled
